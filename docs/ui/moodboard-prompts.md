# Moodboard Prompts (v0)

Use these prompts in DALL·E or similar tools to generate reference images for UI styling. Generate 4 images per direction.

## How to Use
- Aspect ratio: `16:9` and `4:3` mixed.
- Style target: concept art + UI background reference (not finished app screenshots).
- Keep text minimal or none.

## Quick Selection Rubric
Score each direction 1-5 for:
- Theme Fit (fantasy + imagination)
- UI Readability Potential
- GM Density Compatibility
- Player Emotional Resonance
- Distinctiveness vs generic SaaS

Pick top 2 for next iteration.
